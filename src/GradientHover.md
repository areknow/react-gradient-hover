# GradientHover

The main component that creates the interactive gradient hover effect.

## Basic Usage

```jsx
import { GradientHover } from 'react-gradient-hover';

<GradientHover>
  <div style={{ padding: '2rem' }}>
    <h2>Hover over me!</h2>
    <p>The gradient will follow your mouse movement.</p>
  </div>
</GradientHover>
```
